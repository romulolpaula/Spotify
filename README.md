# Projeto Spotify

## Objetivo
O projeto **Spotify**, desenvolvido durante a **Imersão Front-end da Alura**, teve como objetivo replicar uma interface inspirada no Spotify. Ele busca proporcionar prática em desenvolvimento web com HTML, CSS e JavaScript, além de introduzir o uso de dados JSON.

## Tecnologias Utilizadas

- **Linguagens**:
  - **HTML**: Para estruturar a página.
  - **CSS**: Para estilização e design responsivo.
  - **JavaScript**: Para interatividade e manipulação de DOM.
- **Banco de Dados Simulado**:
  - **JSON**: Para armazenar e consumir dados fictícios sobre artistas.

## O que foi Abordado

1. **Criação de Interface Responsiva**:
   - Desenvolvimento de uma interface visualmente similar ao Spotify, explorando estilização com CSS e organização de componentes em HTML.
2. **Interatividade com JavaScript**:
   - Implementação de funcionalidades dinâmicas, como busca e manipulação de elementos DOM para exibir os dados carregados.
3. **Uso de JSON**:
   - Consumo de dados de um arquivo JSON (`artists.json`) contendo informações fictícias de artistas e músicas.

## Estrutura do Projeto

- `index.html`: Estrutura principal da aplicação.
- `style.css`: Arquivo responsável pela estilização da página.
- `script.js` e `search.js`: Scripts JavaScript para funcionalidades e manipulação de DOM.
- `artists.json`: Dados simulados de artistas para exibição na página.
